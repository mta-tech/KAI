# Keyboard Shortcuts Reference

**KAI is designed for keyboard-driven workflows.** Power users rarely need to touch the mouse.

Press `?` anywhere in KAI to see this reference sheet.

---

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Cmd/Ctrl + K` | Open Command Palette | Quick access to all commands |
| `Cmd/Ctrl + /` | Focus Search | Search across all content |
| `?` | Show Keyboard Shortcuts | Display this reference |
| `Esc` | Close Modal/Dialog | Dismiss any open modal |
| `Cmd/Ctrl + ,` | Open Settings | Navigate to settings |

**Pro Tip:** Use the Command Palette (`Cmd/Ctrl + K`) for fastest navigation. It predicts what you need based on context.

## Tips for Power Users

### 1. Learn the Command Palette

The command palette (`Cmd/Ctrl + K`) is your most powerful tool. It:
- Predicts what you need based on context
- Provides fuzzy search
- Shows keyboard hints for all actions

### 2. Use Vim-Style Navigation

If you prefer Vim-style navigation:
- `j` / `k` for down/up
- `h` / `l` for left/right
- `g` then `n` to go to section `n`

### 3. Chain Shortcuts

Combine shortcuts for speed:
- `Cmd/Ctrl + K` → `new` → `Enter` = New chat
- `g` → `q` → `n` = New query in knowledge base

### 4. Use Search Effectively

- `Cmd/Ctrl + /` focuses search globally
- Type anywhere to filter current list
- Use fuzzy matching (e.g., "kb" matches "Knowledge Base")
