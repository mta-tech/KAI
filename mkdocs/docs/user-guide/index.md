## Best Practices

### Context Assets

1. **Start with table descriptions** - Document your most important tables first
2. **Build glossaries incrementally** - Add business terms as you encounter them
3. **Use instructions for consistency** - Define rules for common query patterns
4. **Promote through lifecycle** - Verify assets before publishing

### Querying

1. **Be specific** - Include column names, time periods, and filters
2. **Use interactive mode** - Build complex analyses iteratively
3. **Review generated SQL** - Understand what KAI is doing
4. **Export sessions** - Document your analysis workflows

### Knowledge Management

1. **Curate actively** - Keep glossaries and instructions up to date
2. **Tag consistently** - Use tags to organize and discover assets
3. **Run benchmarks** - Validate asset quality regularly
4. **Monitor telemetry** - Track asset reuse for KPIs
